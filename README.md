## 🔬 First Technical Findings — Week 1

**Scene:** Bad Decision Studio (BDS) — Star Wars-inspired cinematic environment
*(Stormtrooper formation array + Lambda-class Shuttle with animation · 175 actors)*
**Engine:** Unreal Engine 5.4

### Shader Complexity Baseline

**Tool:** `Alt+8` — Shader Complexity View

| Zone | Color | Reading |
|------|-------|---------|
| Platform floor | Red | Heavy — multi-layer material, 6–7 texture samples |
| Left overhead section | Magenta | Exceeds MaxShaderComplexityCount=2000 — severe overdraw suspected |
| Stormtroopers | Green | Efficient — simple shaders |
| Background / sky | Green | Efficient |

**Frame data:** 120 FPS · 8.33ms frame time *(editor viewport — not representative of packaged build)*

**Root finding on floor material:**
- Contains 6–7 Texture Samples in a single material graph
- Macro Texture Variation using 3 separate UV scales (`0.2134`, `0.05341`, `0.002`)
- Each UV scale adds GPU sampling cost — lowest-impact layer (`0.002`) is first optimization candidate

**Planned optimization** *(Phase 3)*:
- Disconnect lowest-impact macro layer and measure delta
- Implement ORM channel packing to reduce sample count
- Document Shader Complexity before/after as first public case study

### Bugs Resolved This Week

All bugs documented in detail in [`TROUBLESHOOTING.md`](./TROUBLESHOOTING.md).

| Bug | Root Cause | Fix |
|-----|-----------|-----|
| Viewport shadow inconsistency | UE 5.7 rendering pipeline mismatch with tutorial | Downgraded to UE 5.4, rebuilt project |
| Paid decal asset required | Color came from material node, not texture | White texture + Multiply node + Constant3Vector (yellow) |
| CineCamera autofocus inactive on rig | Camera not initialized as active | Pilot → Play → Stop → Exit pilot sequence |
| Ghost shadow on moving objects | Shadow Denoiser bug specific to UE 5.4 | `r.Shadow.Denoiser=0` in `Config/DefaultEngine.ini` |

> The goal is not perfection — it is building the habit of asking *why* before asking *how.*
